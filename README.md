## ✦ Who am I?
I'm Augusta — a Cybernetician, Engineer, and Agentic Builder. This website acts as an archive of my writing, open-source work, and thoughts on societal feedback loops and machine-to-human systems architecture. 

## ✦ Tech Stack
This site is engineered with speed, simplicity, and aesthetics in mind.
- **Framework:** [Astro](https://astro.build) (Static Site Generation)
- **Styling:** Vanilla TailwindCSS
- **Interactions:** Custom vanilla JavaScript and GSAP for kinetic topology visualizations

## ✦ Getting Started Locally
To spin this up on your local machine and take a look at the architecture:

```bash
# 1. Install dependencies
npm install

# 2. Start the local dev server
npm run dev
```

Feel free to poke around the `/src/content/articles` structure or admire the custom canvas element in the Hero components!
